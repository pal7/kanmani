# Kanmani · கண்மணி

> Bilingual Tamil-English AI Chat Assistant

A production-quality PWA demonstrating bilingual UX, model routing by language, and SSE streaming — built with React 18, Node/Express, and Azure AI Foundry.

## Quick start

```bash
nvm use          # Node 20
pnpm install
cp server/.env.example server/.env   # fill in your values
pnpm dev
```

See [CLAUDE.md](./CLAUDE.md) for full architecture documentation.
